# Logic for Programming course @ IST

**Project - Star Battle Puzzle Solver**

Goal: Build a Prolog program capable of representing, visualizing, and solving a Star Battle–style logic puzzle. The
project focuses on modeling the puzzle as a matrix of regions, enforcing placement rules, and implementing predicates
that insert stars, validate constraints, and apply simple solving strategies. A hands‑on introduction to logical
reasoning, recursion, and declarative problem‑solving using Prolog.
