# Logic for Programming

> The objectives are that the students acquire knowledge about the components of a logic, and be able to
> carry out proofs in the deductive system. The students will also acquire notions about automatic
> reasoning and will be able to develop programs in the logic programming paradigm.

## Project - *Star Battle* Puzzle Solver

### Goal
Build a Prolog program capable of representing, visualizing, and solving a *Star Battle*–style logic
puzzle. The project focuses on modeling the puzzle as a matrix of regions, enforcing placement rules,
and implementing predicates that insert stars, validate constraints, and apply simple solving
strategies. A hands‑on introduction to logical reasoning, recursion, and declarative problem‑solving
using Prolog.
